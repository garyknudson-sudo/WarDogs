# War Dogs Football Website

Production-ready marketing website for **War Dogs Football** built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Brutalist, high-contrast visual design with aggressive geometry
- Mobile-first responsive layout
- Smooth scroll + section reveal animations
- SEO metadata configured
- Optimized images via `next/image`
- Clear conversion-focused page structure:
  - Hero
  - Pain agitation
  - Benefits
  - Process
  - Testimonials
  - Credibility/About
  - Final CTA
  - Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run start
```
